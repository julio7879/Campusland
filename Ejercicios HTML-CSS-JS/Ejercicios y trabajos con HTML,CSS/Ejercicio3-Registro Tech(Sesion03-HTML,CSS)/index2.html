Programa de aprendizaje nivel bajo en HTML

Este programa está diseñado para estudiantes que inician desde cero y quieren aprender a construir páginas web paso a paso.

Algunas cosas que aprenderas en el curso de aprendizaje

  1. aprender etiquetas de html
  2. aprender atributos de etiquetas
  3. aprender a crear formularios

¿Qué veremos en esta sesión?

  • Separadores para organizar el contenido.
  • Listas para mostrar ideas o pasos.
  • Tablas para presentar información ordenada.
  • Formularios para recolectar datos del usuario.

Pasos para participar

  1. Leer la información del programa.
  2. Revisar el horario disponible.
  3. Completar el formulario de registro.
  4. Confirmar los datos antes de enviar.
